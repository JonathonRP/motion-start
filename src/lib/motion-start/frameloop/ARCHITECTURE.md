## Delta Capping

```
Normal Operation:
Frame 1: delta = 16.67ms  вң“
Frame 2: delta = 16.67ms  вң“
Frame 3: delta = 16.67ms  вң“

Tab Backgrounded:
Frame N: delta = 5000ms   вң— (capped to 40ms) вң“

Why?
- Prevents animations from jumping too far
- Avoids "spiral of death" (slow вҶ’ slower вҶ’ slowest)
- Maintains smooth animation even under stress

Formula:
delta = Math.max(
  Math.min(
    timestamp - prevTimestamp,
    40  // maxElapsed
  ),
  1   // minimum
)
```
